Challenge Results: Calculate and Predict CLV per Region¶

Using the 'superstore_final_dataset.csv', from Kaggle, we will implement the following analysis:

  • Load and view dataset
  • Perform an initial exploratory data analysis
  • Perform feature engineering to groupby relevant features and calculate Customer Lifetime Value
  • Create train, validate and test datasets
  • Select an appropriate regession model using CLV as the target variable
  • Run cross validation and a hyperparameter grid search to faciliate optimising the model training
  • Test predictive model against the test dataset

Feature Engineering¶

Create CLV target variable and assemble a dataset for data and predictive analysis

We are interested in creating a predictive algortihm for geographic locations, in this instance for "State". Hence we'll calculate CLV by deriving variables for Customer Tenure, Frequency of Purchases and Order Value. We'll then drop these feartures and see if CLV can be predicted by geographical location, hence allow us to consider scenario options for forecasting marketing spend, and the return, in different locations.

# First, convert Order_Date and Ship_Date columns to datetime objects
df['Order_Date'] = pd.to_datetime(df['Order_Date'])
df['Ship_Date'] = pd.to_datetime(df['Ship_Date'])

# We're aiming to create a predictive algorithm for CLV by 'Customer_ID', 'Segment', 'Region', 'State'
# Hence we need to calculate CLV by that grouping, so we can use CLV as our target training metric
# First we will assemble the dataset from the original superstore dataset:

# Calculate customer tenure (time between first purchase date and last purchase date)
customer_tenure = df.groupby(['Customer_ID', 'Segment', 'Region', 'State'])['Order_Date'].agg(lambda x: (x.max() - x.min()).days)

# Calculate frequency of purchases (number of orders per customer)
frequency_of_purchases = df.groupby(['Customer_ID', 'Segment', 'Region', 'State'])['Order_ID'].nunique()

# Calculate average order value (total sales amount divided by number of orders)
average_order_value = df.groupby(['Customer_ID', 'Segment', 'Region', 'State'])['Sales'].sum() / df.groupby(['Customer_ID', 'Segment', 'Region', 'State'])['Order_ID'].nunique()

# Combine the calculated metrics into a DataFrame
clv_data = pd.DataFrame({
    'Customer_Tenure': customer_tenure,
    'Frequency_of_Purchases': frequency_of_purchases,
    'Average_Order_Value': average_order_value
})

# Now we will Calculate the CLV approximation for the 'Customer_ID', 'Segment', 'Region' grouping

# For simplicity, let's assume that the customer's lifetime is represented by their tenure
# CLV = Average_Order_Value * Frequency_of_Purchases * Customer_Tenure
clv_data['CLV'] = clv_data['Average_Order_Value'] * clv_data['Frequency_of_Purchases'] * clv_data['Customer_Tenure']

import pandas as pd

df = clv_data_filtered.reset_index()

# Function to eliminate outliers using Z-score
def remove_outliers_zscore(df, column, threshold=3):
    z_scores = (df[column] - df[column].mean()) / df[column].std()
    df_filtered = df[(z_scores.abs() < threshold)]
    return df_filtered

# Remove outliers from the 'CLV' column using Z-score method

import pandas as pd

from sklearn.metrics import mean_squared_error

# Define the XGBoost regressor
xgb_regressor = xgb.XGBRegressor()

# Define hyperparameters grid
param_grid = {
    'n_estimators': [100, 200, 300],
    'learning_rate': [0.05, 0.1, 0.2],
    'max_depth': [3, 4, 5],
    'min_child_weight': [1, 3, 5],
